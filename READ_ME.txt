The code consists of 5 main parts
1. anserini_eval_install.py: for installing Anserini and all nescessary dependencies
2. indexing.py: indexing using Anserini and Calculating baseline
3. query_operations.py: performing all queries operations and write the retrieval outputs to .txt file
4. evaluation.py: evaluate the .txt file
5. topic_clean.py: clean out topics.backgroundlinking19.txt
